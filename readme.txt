== Description ==
各カスタム投稿タイプにカスタムバルクアクションを追加します。

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-bulk-action` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Custom Bulk Action screen to configure the plugin.

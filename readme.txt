== Description ==
各カスタム投稿タイプにカスタムバルクアクションを追加します。

== Installation ==
1. プラグインファイルを `wp-content/plugins/` ディレクトリにアップロードします。
2. WordPress の「プラグイン」メニューからプラグインを有効化します。
